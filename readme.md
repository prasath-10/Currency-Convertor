# Currency Converter App

A simple **currency converter** built with **React**, **Tailwind CSS**, and **Vite**.  
This app allows users to convert currencies in real-time using exchange rates from a free API.

---

## Features

- Convert between multiple currencies (USD, INR, EUR, etc.)
- Swap "from" and "to" currencies
- Show amounts rounded to **2 decimal places**
- Responsive design using **Tailwind CSS**
- Background image for a modern look
- Loading state while fetching exchange rates

---

## Tech Stack

- **Frontend:** React.js, Tailwind CSS, Vite
- **Hooks:** `useState`, `useEffect`, custom `useCurrencyInfo` hook
- **API:** [ExchangeRate-API](https://api.exchangerate-api.com/)  
- **Bundler:** Vite
